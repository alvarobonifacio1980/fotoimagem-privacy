Política de Privacidade – FotoImagem

Última atualização: 30/12/2025

O aplicativo FotoImagem é um aplicativo móvel que atua como um cliente de acesso ao serviço web disponível em:
https://fotoimagem.streamlit.app/

Coleta de dados

O aplicativo FotoImagem não coleta, armazena ou compartilha dados pessoais diretamente.

Todo o conteúdo exibido no aplicativo é fornecido por meio de uma interface web (Streamlit), acessada dentro do aplicativo através de um WebView. Qualquer informação eventualmente inserida pelo usuário ocorre diretamente no ambiente web e segue as políticas da plataforma utilizada para hospedar o serviço.

Uso de informações

O aplicativo não utiliza mecanismos próprios de rastreamento, análise de comportamento, publicidade ou monetização por anúncios.

Cookies e tecnologias semelhantes

O aplicativo em si não utiliza cookies. No entanto, a aplicação web acessada pode utilizar cookies ou dados temporários estritamente necessários para o funcionamento técnico do serviço.

Compartilhamento de dados

Nenhuma informação pessoal é compartilhada pelo aplicativo.

Segurança

O acesso ao serviço é realizado por meio de conexão segura (HTTPS), garantindo a proteção dos dados durante a transmissão.

Privacidade de crianças

O aplicativo não é direcionado a crianças menores de 13 anos e não coleta intencionalmente informações pessoais de crianças.

Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente. Recomenda-se que os usuários revisem este documento ocasionalmente para se manterem informados sobre eventuais mudanças.
